class PMVersion

? ^ 

Version number.

Represents a specific version number. It provides mechanisms for version comparison.
An PMVersion is composed of:

  • A major number.
  • A minor number.
  • A patchlevel.
  • A buildlevel.

You can construct, destruct, copy, and assign objects of this class.
